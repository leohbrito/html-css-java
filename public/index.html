Planning Poker - Sala:

🎯 Planning Poker

Sistema de votação ágil usando Fibonacci para estimativas de story points.

Funcionalidades:

  • ✅ Votação em tempo real
  • ✅ Sequência Fibonacci (0-34)
  • ✅ Estatísticas automáticas
  • ✅ Modo espectador
  • ✅ Timer com auto-revelação
  • ✅ Múltiplos decks
  • ✅ Controles de host

Como usar:

  1. Entre na sala com seu nome
  2. Escolha uma carta para votar
  3. Aguarde todos votarem
  4. O host revela os votos
  5. Analise as estatísticas
  6. Resete para nova votação

Atalhos de teclado:

  • 1-9 - Votar com cartas
  • 0 - Votar zero
  • ? - Votar incerteza
  • R - Revelar (host)
  • C - Resetar (host)
  • S - Copiar link da sala
